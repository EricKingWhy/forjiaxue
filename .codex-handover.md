## 关键技术约束

| 约束 | 值 |
|------|-----|
| 照片格式 | jpg/jpeg/png/webp, ≤10MB |
| 音乐格式 | mp3/m4a/wav, ≤20MB |
| EXIF | 必须移除 |
| IP存储 | 仅哈希（SHA-256），不存原始IP |
| 数据库路径 | `backend/data/app.db` |
| Uploads路径 | `backend/uploads/original/webp/thumb/music/particle_map/` |
| 粒子分层 | high=60000, medium=30000, low=12000 |
| Bloom | 低性能设备自动禁用 |
| 前端路由 | `/ForJiaXue` 固定路径 |
